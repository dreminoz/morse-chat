# 모스 포켓

문구를 기기에 저장하고 모스부호 진동으로 재생하며, 무작위 알파벳 훈련을 할 수 있는 설치형 모바일 웹앱입니다.

## 실행

프로젝트 최상위 폴더에서 MORSE CHAT 서버를 실행한 뒤 휴대폰으로 접속하세요.

```powershell
node server/server.js
```

같은 Wi-Fi의 휴대폰에서 `http://컴퓨터-IP:8787`으로 접속하면 친구 대화, 우주 시그널, 랜덤 시그널이 실제로 연결됩니다.

> 웹 진동 API는 주로 Android Chrome에서 지원됩니다. iPhone Safari는 진동 재생을 지원하지 않습니다.
